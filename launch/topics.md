# Topics

PH lets you pick up to **four** topics. Topics control which "best of" rankings the listing appears in and influence the algorithm's audience match.

## Picks

1. **Developer Tools**
2. **Open Source**
3. **Artificial Intelligence**
4. **GitHub**

## Why these

- **Developer Tools** is the home topic — non-negotiable. Largest dev-audience pool on PH.
- **Open Source** signals "this is freely available" and pulls in the OSS-curious browsers.
- **Artificial Intelligence** is where the MCP hook lands. Without it, the AI-shopping audience won't see this.
- **GitHub** maps to the framework's distribution (GitHub repo, GitHub releases, etc.). Also pulls in the "what's hot on GitHub" cohort.

## Rejected

- **SaaS** — wrong audience. Keryx is a framework, not a hosted product.
- **Productivity** — too generic; would dilute the dev-tools signal.
- **API** — too narrow; Developer Tools already covers it.
- **TypeScript** — not actually a topic on PH (it's a tag in some places but not a topic).
